# Matt Alter's Serve Robotics Live Map


## The project has two parts
First, run `npm install` to install all project packages

### Server
- Run the server from `/server` with `npm start`

### Front-end
- Built with React, it uses open streetmaps to generate the map.  
- Uses React Leaflet to generate a popover visible on hover of each robot to show its number and geo positioning
Run using `npm start`
